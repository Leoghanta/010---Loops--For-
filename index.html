Workshop 10: Using For Loops in JavaScript

Workshop 10

In JavaScript, a for loop is used to repeat a block of code a certain number of times. It's very useful when you know how many times you want to loop over a block of code.

Syntax of a `for` Loop

The basic syntax of a `for` loop looks like this:


for (initialization; condition; increment/decrement) {
    // Code to be executed in each iteration
}

Here’s what each part means:

  • Initialization: This sets the starting value of the loop variable.
  • Condition: The loop continues as long as this condition is true.
  • Increment/Decrement: This changes the loop variable after each iteration.

Example 1: Counting from 1 to 5


for (let i = 1; i <= 5; i++) {
    console.log(i);
}
// Output: 1, 2, 3, 4, 5

This loop starts at 1 and increments by 1 in each iteration, stopping when the value of i exceeds 5.

Example 2: Looping Through an Array

Sometimes you’ll want to iterate over the elements of an array. A `for` loop can help you do this efficiently.

Example: Displaying an Array of Colours


const colours = ["Red", "Green", "Blue", "Yellow"];
for (let i = 0; i < colours.length; i++) {
    console.log(colours[i]);
}
// Output: Red, Green, Blue, Yellow

Example 3: Summing Numbers in an Array

Let's take an array of numbers and use a `for` loop to sum the numbers.


const numbers = [10, 20, 30, 40];
let sum = 0;
for (let i = 0; i < numbers.length; i++) {
    sum += numbers[i]; // Add each number to the sum
}
console.log(sum);
// Output: 100
